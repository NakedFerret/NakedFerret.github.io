Common Limitations

SimplePass was designed to stand out from the rest of the pack. The best way to illustrate this, is to show the common limitations with other password managers.

Copy/Paste

The copy/paste framework is insecure because any installed application can examine the clipboard

In-App Web Browser

Prevents users from inputing passwords in other applications and from using their preferred web browser

Unintuitive

Users are required to learn difficult concepts and user interfaces are not optimized

The Simple Approach

SimplePass aims to address these shortcomings with a new password input system.

Simple

Users can think of their password in terms of Vaults and Accounts. Passwords are stored in Accounts. Accounts are stored in vaults.

Ubiquitous

The new input system a touch away no matter where you are in your device

Unobtrusive

The input system is brought up by touching each side of the device's screen. This unique gesture prevents unintentional touches.
